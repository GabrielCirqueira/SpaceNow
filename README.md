# SpaceNow

SpaceNow é uma plataforma temática de exploração espacial que apresenta dados reais fornecidos de fenômenos cósmicos pela NASA. Este repositório contém a base do sistema (front-end) — com layout, páginas e componentes prontos — para que você desenvolva a aplicação de visualização e exploração desses dados.

O projeto já inclui conveniências e boas práticas para acelerar o desenvolvimento e manter consistência entre as telas e componentes.
